# Dragon's Curse

Dragon's Curse is a 2D action-adventure game featuring multiple environments, challenging enemies, and progressive difficulty levels. Players explore dangerous worlds, survive enemy encounters, and complete increasingly difficult stages while experiencing immersive gameplay and stylized environments.

# Project Structure

Recommended project structure:

```bash
Assets/
ProjectSettings/
Packages/
README.md
```

---

# Repository Checklist

Your Unity repository should include:

✅ Assets/
✅ Packages/
✅ ProjectSettings/
✅ README.md

Your repository should NOT include:

❌ Library/
❌ Temp/
❌ Logs/
❌ Obj/
❌ Builds/ (optional)

These folders are automatically generated by Unity and make repositories unnecessarily large.

# Installation

1. Clone the repository
2. Open the project in Unity
3. Load the main scene
4. Press Play to run the game
